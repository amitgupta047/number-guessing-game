# 🎯 Number Guessing Game

An interactive number guessing game built using HTML, CSS, and JavaScript, featuring an intelligent AI hint system.

🚀 Features
🔢 Random number generation (1–100)
🎯 Limited attempts (10 guesses)
🤖 AI Hint System (Binary Search based)
📉 Displays previous guesses
🔄 Restart game functionality
🧠 How to Play
Enter a number between 1 and 100
Click Submit
Follow hints:
Too High 📈
Too Low 📉
Use AI suggestions to guess efficiently
Win within 10 attempts 🎉
